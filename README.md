# Qual-esse-rem-dio-
POC Android em Kotlin + Jetpack Compose que usa ML Kit Text Recognition (OCR on-device) para identificar o nome de um medicamento a partir de uma foto. Inclui captura via câmera, recorte para melhorar o OCR, pós-processamento para filtrar ruídos e opção de editar o termo antes de pesquisar no navegador.
